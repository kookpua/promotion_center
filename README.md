
# 促销服务
促销服务维护着所有促销信息,核心业务为:
- 促销基础信息
- 设置促销规则
- 设置参与促销的商品

# 架构模式
![avatar](/img/促销中心架构图.png)

# 基本框架搭建
- Net Core3.1
- Autofac
- Swagger
- SqlServer
- Dapper
- Redis
- Docker
- 单元测试

# 开发模式 
- DDD
- RestFul


## 买一送一
- 新增
- 修改
- 删除
- 查询
- 上下架
- 设置活动商品 productid

## 满额送
- 新增
- 修改
- 删除
- 查询
- 上下架
- 设置活动商品
- 设置满额送规则

## 商品
- 根据单个商品查询参与活动信息(商品详情)
- 根据多个商品查询参与活动信息(购物车/下单)
